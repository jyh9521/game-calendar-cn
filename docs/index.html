如何订阅？

📱 Apple 日历（iPhone / iPad / Mac）
  1. 复制上方的订阅地址
  2. 打开 设置 → 日历 → 账户 → 添加账户 → 其他
  3. 选择 添加已订阅的日历
  4. 粘贴订阅地址并存储
📧 Google 日历
  1. 复制上方的订阅地址
  2. 打开 Google 日历
  3. 点击左侧 其他日历 旁的 +
  4. 选择 通过网址，粘贴订阅地址
💼 Outlook
  1. 复制上方的订阅地址
  2. 打开 Outlook → 日历
  3. 点击 添加日历 → 从 Internet 订阅
  4. 粘贴订阅地址并确认
